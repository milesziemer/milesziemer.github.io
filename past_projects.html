Past Projects

HowManyPills App (Summer 2019)

This desktop app was created to give patients who have been perscribed narcotic pills an idea of how many pills people with their similar BMI and age took. I originally coded this in Visual Basic, but put it into C#, believing C# would be more of a learning experience, and would prove more useful in the future. This program uses data from lower extremity surgeries performed at the Anne Arundel Medical Center, and was presented to some of the staff, who implemented the idea into the framework they already used to track perscriptions.

Roomba Programming (Fall 2019)

This is a demonstration of a project I worked on for my robotics class. We used a RaspberryPi and Python to program a Roomba robot. In this video, we used a PID controller, along with laser sensor data to have the robot follow along a wall. Other projects in this course included random walk, using the bumpers on the robot, and docking the robot at the charging station. One big thing about this project that was difficult but proved priceless, was the intricacy and specificity of robot hardware. We had to access each sensor's data directly, as well as it's interactive components. This led to significant bugs and errors, which can be difficult to diagnose. Fortunately, the Roomba has a very specific design architecture that made it more structured to deal with problems. Exposure to this kind of problem solving was extremely valuable to me.

FPGA Programming (Spring 2019)

This is a demonstration of a project I worked on for my Embedded Systems course. We setup an FPGA using software to interface our C code with the components of the FPGA. Specifically, we used the video output of the FPGA, and hooked it up to a monitor, as well as running multiple processors for speed. The C code we wrote iterated and zoomed into the Mandelbrot set. Personally, I enjoyed the mathematical aspect of this project, but mostly was shown the connections between software and hardware that allow complex and/or specific machines to run. Other projects in this course used other aspects of the FPGA board, such as the LED's and 8-bit display. In general, I learned a great deal about how to connect software and hardware, as well as how to design specific hardware to complete a task.

Reinforcement Learning (Spring 2020)

This is a project I did for my Artificial Inteligence course. We implemented value iteration and Q-learning in order to make a Pacman agent that can win the game by itself. The mathematical depth of learning in this way is very interesting and beautiful to me, but it is also fantastic to have seen how we might simplify problems into a form that can be described with such methods. We used Python to code this, and built it on top of the Pacman AI projects from UC Berkley. This video is a demonstration of Q-learning, and is just one of many different implementations/tests of Q-learning and value iteration.

System Verilog (Fall 2020)

This is a demo of my first project for my Advanced Digital Logic course. We created a hex counter in System Verilog and ran it on a simulated FPGA board. We also implemented a testbench for this project to verify the functionality of the counter. Throughout the semester, we will have more complicated and difficult projects, which I am sure will teach me much more about System Verilog. This one gave me an introduction to System Verilog, and its format as well as some insight into how to use it.
