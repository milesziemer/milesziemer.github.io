Current Projects

Project Athena

This is a project for my Machine Learning course. We are designing and building an adversarial machine learning system on top of the Athena framework. The goal is to gain experience creating a machine learning system, and to make sure it can protect against evasion attacks.

Stock Market Prediction

This is my senior design project for CSCE at the University of South Carolina. My team will be building a neural network that attempts to predict the future movement of the stock market in certian time blocks. The predictions and user interface will be implemented using Django, and the project will turn into a web application. In theory, we are aiming to get experience with web app development on all levels, however my group and I are very interested in the development of the neural network, and would very much like to make it work.
